# 🍽️ QR Kodlu Sipariş Yönetimi - Asp.Net Core API + SignalR

**Eğitmen:** Murat Yücedağ  

Uzun süredir üzerinde çalıştığım bu kursu tamamlamış bulunuyorum. Bu projede, **Asp.Net Core 8.0** ile geliştirilmiş admin ve vitrin panelleri bulunan, **SignalR kütüphanesiyle gerçek zamanlı (real-time) işlevler** eklenmiş kapsamlı bir sipariş yönetimi uygulamasını inceledim ve uyguladım.  

> ⚠️ Kursa başlamadan önce Asp.Net Core ile çalışmış olmanız önerilir. Sadece C# bilenler için proje biraz zorlayıcı olabilir.

---

## 🚀 Proje Hakkında
Bu proje kapsamında bir restoran için **QR kodlu sipariş yönetimi sistemi** geliştirdik.  

- **Backend**: ASP.NET Core API  
- **Frontend**: Admin paneli ve vitrin paneli  
- **Real-Time İşlevler**:  
  - Anlık bildirimler  
  - Anlık mesajlaşma  
  - Anlık sepet güncellemeleri  
  - Real-time istatistikler  

Projeyi geliştirirken eğitmenimiz Murat Yücedağ’ın yönlendirmeleri doğrultusunda **SOLID prensipleri**, **Clean Code** kuralları ve **N-Tier mimari**yi uyguladık. Ayrıca rezervasyon işlemleri için kullanıcılarımıza **mail gönderdik**.

---

## 💻 Kullanılan Teknolojiler

- Asp.Net Core 8.0 / API  
- SignalR - Real-time uygulamalar için  
- Swagger - API dokümantasyonu  
- DTO (Data Transfer Object)  
- N-Tier Architecture  
- MSSQL - İlişkili tablolar  
- Entity Framework & LINQ  
- HTML, CSS, Bootstrap, JavaScript, AJAX  
- Alert ve Basket Sistemleri  
- Rezervasyon İşlemleri ve Mail Gönderimi  
- Real-Time Mesajlaşma, Bildirim ve İstatistikler
